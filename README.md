# SCIP: Smart Community Intelligence Platform

SCIP is a next-generation civic management system designed to bridge the gap between citizens and local authorities through transparent, AI-driven governance. Built with a focus on accountability and efficiency, SCIP transforms raw community feedback into actionable intelligence.

## 🌟 Project Vision
SCIP moves away from traditional "ticket-based" systems and adopts a **"Civic Case-File"** identity. Every report is treated as a living document, monitored by AI to ensure it reaches the right department with the correct priority.

## 🚀 Core Features

### For Citizens
- **Smart Reporting**: Submit issues with geolocation, image uploads, and real-time AI categorization.
- **Live Case Tracking**: A visual timeline of resolution progress, from submission to final verification.
- **AI Community Assistant**: A 24/7 chat interface to ask about complaint trends, risk areas, or legal filing procedures.
- **Feedback Loop**: Rate resolution quality to help the system improve departmental accountability.

### For Administrators
- **Executive Reports**: Automated high-level summaries generated by Gemini for leadership review.
- **AI Priority Ranking**: Automatic scoring of complaints based on urgency, risk, and community impact.
- **Resource Allocation**: Manage ground teams and authorize deployments based on AI-suggested load balancing.
- **Risk Assessment**: A heat-map style view of community risk scores per area, identifying critical infrastructure needs.

## 🧠 AI Engine: Gemini Integration
SCIP leverages the **Google Gemini (1.5/2.5 Flash)** models via the Google ADK (Agent Development Kit) to power its intelligence layer:

1.  **Agentic Workflows**:
    - **Router Agent**: Analyzes user queries and delegates tasks to specialized sub-agents (Foodie, Navigation, Budget, etc.).
    - **Iterative Planner**: Refines community action plans based on constraints like travel time or budget.
2.  **Automated Analysis**:
    - **Predictive Categorization**: Detects if a complaint belongs to Water, Electricity, or Waste departments based on text description.
    - **Priority Scoring**: Assigns 'Critical', 'High', or 'Medium' levels by assessing public safety risks.
3.  **Insight Generation**: Summarizes thousands of data points into "Executive Recommendations" for city officials.

## 🎨 Design System
The platform utilizes a unique **"Civic Case-File"** aesthetic:
- **Typography**: Uses `Zilla Slab` for headings to evoke a formal, document-heavy feel, paired with monospace fonts for data points.
- **Color Palette**: 
    - `Paper`: Off-white background (#fdfcf8).
    - `Ink`: Deep navy for text (#10182a).
    - `Brass`: A sophisticated gold/brown for AI-highlighted elements.
- **Visual Effects**: Subtle grid backgrounds, glassmorphism (backdrop-blur), and "Gemini Glow" radial gradients to highlight AI-driven sections.

## 🛠 Technical Stack

### Frontend
- **Framework**: React 18+ with **React Router v7** (Framework Mode).
- **Styling**: Tailwind CSS for layout, custom CSS for the identity system, and **Framer Motion** for fluid UI transitions.
- **Data Visualization**: **Recharts** for categorical breakdowns and trend analysis.
- **Icons**: Lucide React.
- **State Management**: React Hooks + Axios for resilient API communication.

### Backend (AI & Agents)
- **Engine**: Node.js / Python integration.
- **AI SDK**: `@google/genai` and `google.adk`.
- **Agents**: Sequential and Parallel agentic workflows for complex query routing.

## 🛠 Setup & Installation

1.  **Clone the repository**:
    ```bash
    git clone <https://github.com/your-repo/SCIP.git>
    ```

2.  **Frontend Setup**:
    ```bash
    cd SCIP/frontend
    npm install
    npm run dev
    ```

3.  **Backend AI Setup**:
    - Create a `.env` file in the backend directory.
    - Add your `GEMINI_API_KEY`.
    - Install Python dependencies (if using the ADK agents):
    ```bash
    pip install google-adk python-dotenv
    ```
